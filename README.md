#File Integrity Monitor (FIM)
Simple tool to detect file changes in a directory using SHA256 hashing.
##Usage
```bash
python fim.py
```
Enter Directory path to scan
Tool saves hashes to hashes.json
Run again and compare to detect changes
##Features
SHA256 Hashing - Secure File Hashing
Change DEtection - Identifies modified, new, and deleted files
Persistent Storage - Saves hashes in JSON format
Directory Recursion -Scans all subdirectories
##Installation
No dependencied required
Standard Python 3
